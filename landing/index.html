How it works

Three things, one conversation.

No registration page. No dashboard. No pricing tiers to compare. You request a Project Pass, you open it in Claude, you describe what to build.

i.

Request a PAP

Drop a message on Telegram and ask for a Project API Pass. We send back a single URL — that's your project. Treat it like a key to your own workshop.

ii.

Open it in Claude

Paste the URL into Claude.ai, Claude Desktop, or click the Claude for Chrome icon while on the page. Claude reads the welcome screen and knows exactly what to do.

iii.

Describe and ship

Tell Claude what you want — an editorial about your trip, a PWA for tracking habits, a multiplayer game. It builds, versions, and publishes. Live URL in 5–30 minutes.
